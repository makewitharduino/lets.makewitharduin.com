    Let's Make With Arduino!

    Arduinoで遊んでみよう！

    Arduinoと少しの電子部品があれば、いろいろなサンプルにチャレンジできます。

    サンプルを通して、電子工作とプログラミングの基礎がわかれば、自分なりにリミックスして遊んでみてください。
    さあ、はじめよう！

    電子工作のサンプルのほか、Arduinoをプログラミングするいろいろなツールも紹介しています。

    mode_edit

    BlocklyDuino
    (Web App)

    Scratchのように、ブロックを組み合わせて、ArduinoのプログラミングができるオンラインWebエディタです。

    get_app

    BlocklyDuino
    (Chrome App)

    BlocklyDuino(Web App)のChrome App版です。

    オフラインで動作するので、インターネットがなくても、プログラミングが可能です。

    extension

    Scratch Extension
    (Scratio)

    Scratch2.0 オフラインエディターからArduinoをプログラミングできる拡張機能です。

    このツールがあれば、Scratchの中に、Arduinoと連携したプログラムを作成することができます。
